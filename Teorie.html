• Metoda analitică
      Pentru separarea analitică a soluţiilor vor fi folosite proprietăţile derivatei.
      Dacă soluţiile ecuaţiei f´(x)=0 pot fi uşor calculate, atunci, pentru a separa soluţiile f(x)=0, este necesar:
      1. să se determine soluţiile distincte a ≤ x1 ≤ x2 ≤ ..., ≤ xn ≤ b ale ecuaţiei f´(x)=0;
      2. considerînd a = x0 şi b = xn+1, să se calculeze valorile f(x0), f(x1), ..., f(xn+1).
      Segmentele[xi, xi+1], i = 0, ..., n, pentru care f(xi) × f(xi+1) < 0 vor conţine cîte cel puţin o soluţie a ecuaţiei f(x)=0.

    • Metoda coardelor
      1. Algoritmul de calcul pentru un număr prestabilit n de aproximări succesive:
        1. Pasul 1. Determinarea extremităţii fixe e şi a aproximării x0:
          c ⇐ a - [f(a) / ( f(b)- f(a) )]* (b - a)
          dacă f(c) × f(a) < 0, atunci e ⇐ a, x0 ⇐ b, altfel e ⇐ b, x0 ⇐ a; i ⇐ 0.
        2. Pasul 2. Calculul x(i+1) conform formulei x(i+1) = x(i) - [f( x(i) ) / ( f(e)- f( x(i) ) )]* ( e - x(i) )
        3. Pasul 3. Dacă i + 1 = n, atunci soluţia calculată x ⇐ x(i). SFÎRŞIT.
          În caz contrar, i ⇐ i+1 şi se revine la pasul 2.
      2. Algoritmul de calcul pentru o exactitate ε dată:
        1. Pasul 1.Determinarea extremităţii fixe e şi a aproximării x0:

          dacă f(c) × f(a) < 0, atunci e ⇐ a, x0 ⇐ b, altfel e ⇐ b, x0 ⇐ a; i ⇐ 0.
        2. Pasul 2. Calculul x(i+1) conform formulei: x(i+1) = x(i) - [f( x(i) ) / ( f(e)- f( x(i) ) )]* ( e - x(i) )

    • Metoda Newton
      1. Algoritmul de calcul pentru un număr dat de aproximări succesive:
        1. Pasul 1. Determinarea aproximării iniţiale x0: c ⇐ a - [f(a) / ( f(b)- f(a) )]* (b - a)
          dacă f(c) × f(a) < 0, atunci x0 ⇐ a, altfel x0 ⇐ b; i ⇐ 0.
        2. Pasul 2. Se calculează x(i+1) conform formulei x(i+1)=x(i) - f(xi)/f'(xi)
        3. Pasul 3. Dacă i+1 = n, atunci soluţia calculată x ⇐ x(i+1). SFÎRŞIT.
          În caz contrar, i ⇐ i+1, apoi se revine la pasul 2.
      2. Algoritmul de calcul pentru o exactitate ε dată:
        1. Pasul 1. Determinarea aproximării iniţiale x0: c ⇐ a - [f(a) / ( f(b)- f(a) )]* (b - a)
          dacă f(c) × f(a) < 0, atunci x0 ⇐ a, altfel x0 ⇐ b; i ⇐ 0.
        2. Pasul 2. Se calculează x(i+1) conform formulei x(i+1)=x(i) - f(xi)/f'(xi)
        3. Pasul 3. Dacă [ M(2) / 2m(1) ] *[ x(i+1) - x(i) ]^2 < ε, atunci soluţia calculată x ⇐ xi+1. SFÎRŞIT.
          În caz contrar, i ⇐ i+1 şi se revine la pasul 2.
